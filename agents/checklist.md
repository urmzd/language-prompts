# Guide Generation Checklist (Use Before Delivering)

## Structure & Organization
- [ ] **Communication-first:** Guide emphasizes practical phrases over tourism tips
- [ ] **Scenario-based:** Content organized by situation (greetings, restaurants, hotels, shopping, transport, emergencies)
- [ ] **Pattern phrases included:** Reusable templates with `[brackets]` for customizable slots
- [ ] **Chapter structure:** Numbered files (00-cover.md, 01-core-phrases.md, etc.)

## Phrase Formatting
- [ ] **Tables for all phrases:** No inline comma-separated phrase lists
- [ ] **Four-column format:** `English | Target Language | Transliteration | Notes`
- [ ] **Politeness tiers:** Casual, Polite, Very Polite options where relevant
- [ ] **Regional variations:** Note differences (e.g., Tokyo vs Osaka) where they affect communication

## Required Phrase Categories
- [ ] **Core survival & politeness:** Yes/no, thank you, sorry, please, help (50+ phrases)
- [ ] **Greetings & small talk:** Introductions, meeting people (30+ phrases)
- [ ] **Shopping:** Sizes, payment, trying on, returns (50+ phrases)
- [ ] **Restaurants:** Ordering, dietary needs, payment, etiquette (50+ phrases)
- [ ] **Hotels:** Check-in/out, requests, problems (50+ phrases)
- [ ] **Transportation:** Tickets, platforms, taxis, buses (50+ phrases)
- [ ] **Directions:** Asking for directions, landmarks, getting unlost (30+ phrases)
- [ ] **Emergencies:** Medical, police, lost items, urgent help (30+ phrases)

## Pattern Phrases Required
- [ ] **Request patterns:** [thing] please, Please [action], Could you [action]?
- [ ] **Permission patterns:** May I [action]?, Is it okay to [action]?
- [ ] **Question patterns:** Where is [place]?, Do you have [thing]?, How much is [thing]?
- [ ] **Preference patterns:** I don't eat [food], Without [ingredient] please

## Quality Checks
- [ ] **Communication focus:** Phrases for interacting with people, not sightseeing tips
- [ ] **Practical & high-frequency:** Tourist-relevant, commonly used phrases
- [ ] **Scannable format:** Brief explanations, dense phrase tables
- [ ] **Markdown valid:** Tables render correctly, headings properly nested
- [ ] **No placeholders/TBDs:** All sections complete

## Cultural Context (Where It Affects Communication)
- [ ] **Etiquette notes:** Include where they affect phrase choice
- [ ] **Formality guidance:** When to use casual vs polite vs very polite
- [ ] **Regional differences:** Note dialect/regional variations in key phrases
